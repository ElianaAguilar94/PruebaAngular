# Prueba Angular

Se desarrolla una aplicación la cual presenta un catálogo de productos, el cual se obtiene de una API, y brinda la opción de radicar un pedido de un producto a través de un formulario, cuyos datos del pedido persisten localmente, de modo que se puedan consultar los pedidos generados y filtrarlos por ID.

# Instalar dependencias
=> npm i --save

# Correr proyecto
=> npm start

# Desarrollado con
- Angular CLI: 7.3.9
